```html
<!-- Theme flash prevention reads localStorage('theme') and adds .dark class -->
<script is:inline>
  (function() {
    var stored = localStorage.getItem('theme');
    var prefersDark = window.matchMedia('(prefers-color-scheme: dark)').matches;
    var theme = stored === 'dark' || stored === 'light' ? stored
      : (stored === 'system' || !stored) ? (prefersDark ? 'dark' : 'light') : 'light';
    if (theme === 'dark') document.documentElement.classList.add('dark');
  })();
</script>
```

**IconWrapper.tsx** — React component for Phosphor duotone icons in React islands:

**ThemeToggle.tsx** — React island with `client:load`:
- Import Sun, Moon, Monitor from `@phosphor-icons/react`
- State: `mode` — `'light' | 'dark' | 'system'`
- On mount (`useEffect`): read `localStorage.getItem('theme')`. If `'dark'` or `'light'`, use that. If `'system'` or null, set `'system'`. Sync with actual DOM state.
- `cycle()` function: light -> dark -> system -> light
  - For each mode: update `localStorage.setItem('theme', mode)`
  - For `'light'`: remove `.dark` from `documentElement`
  - For `'dark'`: add `.dark` to `documentElement`
  - For `'system'`: check `matchMedia('(prefers-color-scheme: dark)')` — add/remove `.dark` accordingly
- Dispatch a custom event `'theme-changed'` after each toggle so Nav can update logo
- Render: ghost-style button, 40x40px, centered flex, `aria-label="Toggle theme"`, rounded
  - Show Sun icon when mode is light, Moon when dark, Monitor when system (all 20px, duotone)
  - Use Tailwind classes: `w-10 h-10 flex items-center justify-center rounded-lg hover:bg-muted transition-colors duration-150`

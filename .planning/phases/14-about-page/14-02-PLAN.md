Start the Astro dev server (`npx astro dev`), then use Playwright to capture full-page screenshots of the About page at three viewport sizes:

1. Desktop: 1920x1080 — save as `about-desktop.png`
2. Tablet: 768x1024 — save as `about-tablet.png`
3. Mobile: 390x844 — save as `about-mobile.png`

Use the MCP Playwright browser tool if available, otherwise use a quick inline Playwright script:
```bash
npx playwright install chromium 2>/dev/null
node -e "
const { chromium } = require('playwright');
(async () => {
  const browser = await chromium.launch();
  const viewports = [
    { name: 'desktop', width: 1920, height: 1080 },
    { name: 'tablet', width: 768, height: 1024 },
    { name: 'mobile', width: 390, height: 844 },
  ];
  for (const vp of viewports) {
    const page = await browser.newPage({ viewport: { width: vp.width, height: vp.height } });
    await page.goto('http://localhost:4321/about', { waitUntil: 'networkidle' });
    await page.waitForTimeout(1000);
    await page.screenshot({ path: 'about-' + vp.name + '.png', fullPage: true });
  }
  await browser.close();
})();
"
```

Save screenshots in the project root for easy inspection.

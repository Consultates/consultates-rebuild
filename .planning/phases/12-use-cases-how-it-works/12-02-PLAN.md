  <action>
Create a React island component with the following structure:

**Layout (desktop):**
- 3-column grid (grid-cols-3) with max-w-[1200px], centered
- Each column: 48px circle at top (centered), title below, description below
- Connecting line between circles: absolute positioned, top 24px, spanning between circle centers
- Line uses gradient: `linear-gradient(90deg, var(--border), var(--primary), var(--border))`

**Layout (mobile < 768px):**
- Single column, vertical layout
- Circles stacked vertically with vertical connecting line between them
- Connecting line hidden or converted to vertical

**Step circles:**
- 48px width/height, border-radius 50%
- Initial state: 1px border var(--border), transparent fill
- Number inside: font-mono, centered, e.g. "01", "02", "03"
- Filled state: background var(--primary), text white, border var(--primary)
- Pulse: scale(1) -> scale(1.15) -> scale(1) CSS transition when filling

**Content per step:**
- Title: font-serif, var(--foreground), ~1.125rem
- Description: font-sans, var(--muted-foreground), max-width 280px, centered
- Initial state: opacity 0, translateY(12px)
- Revealed state: opacity 1, translateY(0), transition 400ms ease-out

**Animation sequence (triggered by useInView, ref on container):**
1. Line draws: CSS width transition from 0% to 100% over ~800ms
2. At 33% line progress (~270ms): Circle 1 fills + pulses, then content 1 reveals (200ms delay)
3. At 66% line progress (~540ms): Circle 2 fills + pulses, then content 2 reveals
4. At 100% line progress (~800ms): Circle 3 fills + pulses, then content 3 reveals

Use useState for animation phase tracking. useEffect with setTimeout chain for sequencing (same pattern as StrokeDrawStats.tsx).

**Reduced motion (RDES-33):**
- Use useReducedMotion() from framer-motion
- If reduced motion: all circles filled, all content visible, line at full width — no animation
- Render everything in final state immediately

**CTA below the grid:**
- Centered, uses `<a>` with className "btn-alive" linking to TidyCal
- Text: "Book a Free Call"
- Appears after step 3 content reveals (fade in)

**Step content (from PRD Section 7):**
- Step 01: "We talk" / "Tell me where you are and what you're trying to figure out. No preparation needed."
- Step 02: "You get a plan" / "Specific to your business, your team, and your goals. Not a template."
- Step 03: "We do the work together" / "Coaching, training, or hands-on leadership. One accountable team, start to finish."

Do NOT use GSAP — use pure CSS transitions + React state + setTimeout. This avoids conflict with the page-level GSAP scroll-snap system.

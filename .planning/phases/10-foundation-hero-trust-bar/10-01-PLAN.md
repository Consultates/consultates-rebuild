Key points:
- Uses `var(--background)` not `var(--bg)` -- the project uses `--background` as the Tailwind-mapped token (see global.css `:root`)
- Uses `var(--muted-foreground)` not `var(--fg-muted)` -- same reason
- Uses `var(--border)` which exists in both systems
- Font size is hardcoded `0.8125rem` (13px) matching `--text-xs` from design system
- Fallback values on `--content-max` and `--content-px` in case variables aren't loaded yet
- No `hidden lg:block` -- visible on ALL viewports per RDES-14
- `flex-wrap: wrap` handles mobile gracefully

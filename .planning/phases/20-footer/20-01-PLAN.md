.footer-link {
  position: relative;
  display: inline-block;
  color: var(--footer-link);
  transition: color 0.2s ease;
  text-decoration: none;
}
.footer-link:hover {
  color: var(--footer-link-hover);
}

  content: '';
  position: absolute;
  bottom: -2px;
  left: 0;
  width: 0;

  width: 100%;
}

- Use `var(--stakes-accent)` for the purple underline color (this is the brand purple used in dark sections)
- `display: inline-block` ensures the `::after` pseudo-element width matches the text, not the container
- Height of 1.5px — thinner/lighter than the nav "Book a Free Call" underline per user decision
- The `var(--ease-out)` fallback ensures smooth deceleration

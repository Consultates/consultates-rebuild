  <files>src/components/islands/StrokeDrawStats.tsx, src/styles/global.css</files>
  <action>
Create a new React island `StrokeDrawStats.tsx` that renders three stats with SVG text stroke-draw animation.

**Stroke-draw approach:** Use SVG `<text>` elements with `stroke-dasharray` and `stroke-dashoffset` animation. Each stat number is rendered as SVG text at the `--text-stat` size with:
- `fill: transparent`
- `stroke: var(--stakes-accent)` (2px)
- `stroke-dasharray` set to the total path length
- `stroke-dashoffset` animates from full length to 0, creating the "drawing" effect
- Font: IBM Plex Serif (serif) for the stat numbers

**Implementation details:**
1. Component accepts an array of stat objects: `{ value: string, label: string, source: string }`
2. Use IntersectionObserver (same dual strategy as CountUpStat: MutationObserver on `.scroll-section` for GSAP desktop, IntersectionObserver fallback for mobile)
3. Sequential reveal: title fades in first, then each stat draws in one after another (staggered by ~800ms), then the pull quote fades in last
4. For the 84% stat, KEEP the count-up behavior -- the number counts up while the stroke draws. Use framer-motion `useMotionValue` + `animate` for the count, and CSS animation for the stroke-draw. The other two stats ("5% to 40%" and "12-18 months") are static text with stroke-draw only.
5. Each stat column contains: SVG number (stroke-draw), supporting text below (regular body text), source citation (mono, small, muted)
6. Respect `prefers-reduced-motion`: show all content immediately with no animation, stroke already fully drawn
7. SVG `<text>` elements need `ref` to measure text length for `stroke-dasharray`. Use `useEffect` + `getTotalLength()` on `<text>` element (or estimate with character count * font-size * 0.6 as fallback since `getTotalLength` is only on path elements -- for `<text>`, use `getComputedTextLength()` and multiply by ~3 for stroke perimeter approximation)

**CSS additions to global.css:**
- Add `@keyframes stroke-draw` that animates `stroke-dashoffset` from var(--dash-length) to 0
- Add `.text-stroke-stat` utility: `font-family: var(--font-serif); font-size: var(--text-stat); fill: transparent; stroke: var(--stakes-accent); stroke-width: 2px;`
